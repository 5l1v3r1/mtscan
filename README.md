# Compiling
Install the following packages:

- Code::Blocks IDE
- GTK+ 2 & dependencies
- libssh (-lssh)
- yajl (-lyajl)
- gpsd (-lgps)
- zlib (-lz)

Select "Release" target in Code::Blocks IDE and compile the project.
